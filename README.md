## Installation and setup

You will need `node` and `npm` installed globally on your machine. To visit app go to `http://localhost:3000/`.

- Backend:
  - clone down `https://github.com/jakuubs/UAM-WMI`
  - run `npm install`
  - run `npm start`

- Frontend:
  - clone down this repository
  - run `npm install`
  - run `npm start`
